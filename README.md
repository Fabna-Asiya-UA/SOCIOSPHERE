# SocioSphere – Social Media Web Application

## Project Overview

SocioSphere is a full-stack social media web application where users can create accounts, share posts, follow other users, and interact with content. The application provides secure authentication and a responsive interface for a smooth user experience.

The backend is built using Django and Django REST Framework, while the frontend is developed using React with Vite.

## Live Demo

https://sociosphere-five.vercel.app/

## Features

* User registration and login
* JWT-based authentication
* Create and delete posts
* Follow and unfollow users
* Profile management
* Responsive user interface

## Technologies Used

### Backend

* Python
* Django
* Django REST Framework
* JWT Authentication

### Frontend

* React (Vite)
* Axios
* Tailwind CSS

### Database

* SQLite

## Installation

### Backend Setup

pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

### Frontend Setup

cd frontend
npm install
npm run dev

## Project Structure

backend/ – Django REST API
frontend/ – React application
models/ – Database models
serializers/ – API data serialization
views/ – Business logic
urls/ – API routes
